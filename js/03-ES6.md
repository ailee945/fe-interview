## 列举ES6、ES7、ES8、ES9、ES10、ES11+ 新特性

### ES6

- let 和 const
- Promise
- Class
- 箭头函数
- 函数参数默认值
- 模板字符串
- 解构语法
- 展开语法
  - 构造数组，调用函数时，将数组表达式或者string在语法层面展开
- 对象属性缩写
  - 键名键值相同
  - 函数省略function

### ES7

- [includes()](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Array/includes)
- 指数操作符**

### ES8

- async、await
