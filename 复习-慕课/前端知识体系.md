### 作用域闭包this

const/let的块级作用域

闭包自由变量的查找是在函数定义的地方向上查找变量不是执行的地方

闭包的应用

- 隐藏数据

  ```js
  // 隐藏数据，只展示API
  function creatCache(){
    const cache = {}
    return {
      set(key, value){
        cache[key] = value
      },
      get(key){
        return cache[key]
      }
    }
  }
  const cache = creatCache()
  cache.set('name','key')
  cache.get('name')
  ```
